# web-projectt
ПОЯСНИТЕЛЬНАЯ ЗАПИСКА

MemGen — Генератор мемов
КРАТКОЕ ОПИСАНИЕ
Веб-приложение на Flask для быстрого создания мемов из готовых шаблонов с системой регистрации, избранного, тегов и резервным копированием данных.

ЦЕЛЬ ПРОЕКТА
Создание полноценного веб-приложения с использованием Flask, SQLAlchemy и Bootstrap, демонстрация навыков работы с БД, авторизацией, загрузкой файлов и обработкой изображений.

ОСНОВНЫЕ ХАРАКТЕРИСТИКИ
Характеристика	Значение
Язык	Python 3.11 + Flask
База данных	SQLite + SQLAlchemy
Фронтенд	Bootstrap 5
Авторизация	Flask-Login
Картинки	Pillow

ФУНКЦИОНАЛ
Регистрация и авторизация

Создание мемов (шаблон + текст + теги)

Избранное и теги

Загрузка своих шаблонов

Редактирование профиля

Удаление своих мемов

Бэкапы БД (авто и ручные)

ТЕХНИЧЕСКИЕ ОСОБЕННОСТИ
Создание мема:

Конвертация PNG с прозрачностью в RGB

Автоподбор размера шрифта

Белый текст с чёрной обводкой

Безопасность:

Хэширование паролей

Защита маршрутов

Бэкапы:

Авто-бэкап при создании мема

Восстановление из копии

Хранение 10 последних

КЛЮЧЕВЫЕ АЛГОРИТМЫ
Аутентификация:

text
пользователь вводит логин и пароль
система ищет пользователя в БД
сравнивает хэш введённого пароля с сохранённым
при успехе создаётся сессия
Генерация мема:

text
выбранный шаблон загружается в память
определяется ширина и высота изображения
рассчитывается кегль шрифта (1/12 от ширины)
текст разбивается на строки и центрируется
готовая картинка сохраняется в формате JPEG
Обработка тегов:

text
пользователь вводит теги через пробел
каждый тег ищется в таблице tags
если тег отсутствует → создаётся новый
мем связывается с каждым тегом через MemeTag
Резервное копирование:

text
при создании мема или по кнопке запускается копирование
файл базы данных копируется в папку backups
в имя файла добавляется метка времени
при превышении 10 файлов самые старые удаляются
СОХРАНЯЕМЫЕ ДАННЫЕ
БД: пользователи, шаблоны, мемы, лайки, избранное, теги

Файлы: аватары, шаблоны, созданные мемы, бэкапы

РЕЗУЛЬТАТ
Рабочее приложение с регистрацией, созданием мемов, избранным, тегами, управлением профилем, бэкапами и API.
